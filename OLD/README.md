# OLD (LEGACY)

Este directorio contiene archivos LEGACY de una versión anterior del proyecto. Se conservan únicamente como referencia histórica y contexto durante el refactor. No forman parte del flujo activo ni deben editarse para nuevas funcionalidades.

Referencias principales actuales:
- index.html (login y selección de rol)
- admin-app.html (módulo de administración)
- prospeccion.html (módulo de prospección)

Notas:
- Cualquier cambio funcional debe realizarse en los archivos actuales mencionados arriba.
- Si necesitas consultar una interacción o UI previa, puedes revisar `index old.html` y `script old.js`, ambos etiquetados explícitamente como LEGACY.
